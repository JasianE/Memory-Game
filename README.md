# Memory Game

A browser-based memory card game built as part of The Odin Project. The goal is to match pairs while keeping track of moves and improving recall.

## Overview

This project focuses on building interactive UI with JavaScript and managing state through user interactions. It reinforces concepts like event handling, DOM manipulation, and game logic.

## Tech Stack

Frontend
- HTML
- CSS
- JavaScript

## Features

- Card flipping and matching logic
- Move tracking
- Game reset functionality
- Simple and responsive UI

## How It Works

Cards are shuffled at the start of each game. Players flip two cards at a time to find matching pairs. If the cards match, they remain revealed. If not, they flip back. The game continues until all pairs are matched.

## Project Context

Built as part of The Odin Project curriculum to practice core JavaScript concepts and interactive UI development.
